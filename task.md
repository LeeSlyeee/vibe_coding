# 임무: iOS 앱 업그레이드 기능 및 도봉구청 안내 추가

## 상태

- [x] **완료됨**

## 요약

사용자의 요청에 따라 iOS 앱 내에서도 OCI 웹 버전과 동일한 **프리미엄 업그레이드 기능**을 구현했습니다. 특히 **도봉구청 상담 연계 시 무료 업그레이드** 혜택을 알리는 배너를 모달 내에 포함시켜, 온/오프라인 경험의 일관성을 확보했습니다.

## 주요 변경 사항

1.  **AuthManager 업데이트**:
    - 사용자 등급(`risk_level`) 외에 **프리미엄 상태(`isPremium`)**를 관리하는 로직을 추가했습니다.
    - 로그인 시 백엔드로부터 프리미엄 여부를 받아와 앱 전역(`UserDefaults`)에 동기화합니다.

2.  **MoodCalendarView 개선**:
    - **업그레이드 버튼**: 캘린더 상단에 프리미엄 사용자가 아니거나 위험군이 아닌 경우 노출되는 '✨ Upgrade' 버튼을 추가했습니다.
    - **프리미엄 모달 구현**:
      - 웹 버전과 동일한 디자인 톤앤매너(Tone & Manner)를 적용했습니다.
      - **도봉구청 상담 안내 섹션**을 눈에 띄는 녹색 박스로 배치하여 지역 연계 서비스를 홍보합니다.
      - 기존 9,900원 -> 4,900원 할인 가격 정책을 반영했습니다.
    - **결제 프로세스**: `/api/payment/upgrade` 엔드포인트를 호출하여 즉시 프리미엄 등급으로 전환되는 시뮬레이션 로직을 연결했습니다.

3.  **AppLoginView**:
    - 로그인 응답에서 `is_premium` 필드를 파싱하여 `AuthManager` 상태를 즉시 갱신하도록 수정했습니다.

## 결과

- 이제 iOS 앱 사용자도 캘린더 화면에서 손쉽게 프리미엄 서비스로 업그레이드할 수 있습니다.
- 도봉구청 상담을 완료한 사용자는 앱 내에서 해당 혜택(무료 업그레이드) 자격을 직관적으로 안내받을 수 있습니다.
