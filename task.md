# 임무: OCI 서버 AI 토큰 증설 (최대 1200)

## 상태

- [x] **완료됨**

## 요약

사용자의 추가 요청("토큰 수가 아직 적다, 더 늘려도 되는가?")에 대응하여, **OCI 서버의 부하를 고려하면서도 답변이 끊기지 않도록 토큰 한도를 대폭 상향**했습니다.

## 주요 변경 사항

### `backend/standalone_ai.py`

- **토큰 할당량 2배 증가**:
  - 기본(Default): 400 -> **800 토큰**
  - 장문(Long Input): 600 -> **1200 토큰** (약 A4 용지 절반 분량까지 생성 가능)
- **타임아웃 연장**:
  - 생성 시간이 길어질 것을 대비해 HTTP Timeout을 60초 -> **120초**로 늘려 '서버 응답 없음' 에러를 방지했습니다.
  - **Gemma-2b 모델**은 가벼워서 1200 토큰을 생성해도 OCI 인스턴스에 치명적인 부하를 주지 않으므로 안전합니다.

## 결과

이제 AI는 훨씬 더 길고 풍부한 답변을 제공합니다.
말하다가 중간에 끊기는 현상은 거의 사라질 것입니다.
