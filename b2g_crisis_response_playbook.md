# 마음온(Maum-On) 위기 알림 대응 매뉴얼 (Playbook)

> **문서 유형**: B2G 납품 시 기관 담당자에게 제공하는 현장 운영 매뉴얼
> **대상 독자**: 보건소, 정신건강복지센터, 학교 상담실 담당자
> **작성일**: 2026-03-02
> **버전**: v1.0

---

## 1. 이 매뉴얼의 목적

마음온 앱은 사용자의 감정 기록을 AI로 분석하여 위기 신호를 감지합니다.
**AI 위기 감지는 보조적 수단이며, 100% 정확성을 보장하지 않습니다.**
이 매뉴얼은 AI가 위기 알림을 발송했을 때 현장 담당자가 취해야 할 **구체적인 행동 절차**를 정의합니다.

---

## 2. 위기 등급 분류 체계

마음온은 3단계 위기 분류 시스템을 사용합니다.

|    등급     | 명칭    | 감지 기준 (예시)                             | 알림 대상                       |
| :---------: | :------ | :------------------------------------------- | :------------------------------ |
| **Level 3** | 🔴 위험 | "죽고 싶다", "자살", "자해", "유서" 등       | 보호자 **즉시** 알림 + SOS 모달 |
| **Level 2** | 🟠 주의 | "사라지고 싶다", "살기 싫다", "의미 없다" 등 | 보호자 알림 (SOS 배너)          |
| **Level 1** | 🟡 관심 | "힘들다", "우울하다", "외롭다" 등            | 로그 기록 (알림 미발송)         |

### 통화 시 주의사항

| ✅ 해야 할 말                    | ❌ 하면 안 되는 말                   |
| :------------------------------- | :----------------------------------- |
| "많이 힘드셨나 봐요"             | "그런 생각은 하지 마세요"            |
| "이야기해 주셔서 고마워요"       | "힘내세요" / "긍정적으로 생각하세요" |
| "제가 도울 수 있는 게 있을까요?" | "다른 사람은 더 힘들어요"            |
| "전문가와 이야기해 보시겠어요?"  | "왜 그런 생각을 해요?"               |

## 4. Level 2 (주의) 대응 절차

1. **24시간 이내** 전화 또는 문자로 안부 확인
2. 최근 7일간 감정 온도 추이를 대시보드에서 확인
3. 하락 추세가 3일 이상 지속되면 → **대면 상담 예약 권유**
4. 기록 내용(보호자 대시보드 리포트)을 의료진에게 공유 가능 여부 확인

---

## 5. Level 1 (관심) 대응 절차

1. 별도 긴급 조치 불필요 — AI가 공감 대화로 대응
2. 주간 리포트에서 Level 1 빈도가 **주 5회 이상**이면 → Level 2로 격상하여 대응
3. 월간 리포트 검토 시 참고 사항으로 기록

---

## 6. 오탐(False Positive) 처리

AI 위기 감지는 키워드 기반이므로 오탐이 발생할 수 있습니다.

### 오탐 사례

- "어제 영화에서 자살 장면이 나왔는데 무서웠어" → Level 3 트리거
- "시험 망해서 죽고 싶다 ㅋㅋ" → Level 3 트리거 (관용적 표현)

### 오탐 시 대응

1. 연락 후 오탐임이 확인되면 **"연락드려서 불편하셨을 수 있는데, 혹시 모르니까요"**로 자연스럽게 마무리
2. 오탐이라 하더라도 절대 "AI가 실수했네요"라고 말하지 않기
3. 오탐 사례는 기관 내부 로그에 기록하여 추후 시스템 개선에 활용

---

## 7. 미감지(False Negative) 대응

AI가 감지하지 못하는 위기 상황에 대비합니다.

### 미감지 가능 사례

- 은유적 표현: "이 세상에서 쉬고 싶다", "여행을 떠나고 싶다"
- 갑작스러운 기록 중단 (7일 이상 미기록)
- 감정 온도가 갑자기 매우 높아진 경우 (위장 가능성)

### 대응

1. **7일 이상 미기록** 시 자동으로 보호자에게 "비활동 알림" 발송
2. 담당자는 주 1회 대시보드에서 **미기록 사용자 목록**을 수동 점검
3. 감정 온도 급변(±3도 이상/일) 시 수동 확인

## 8. 기록 및 보고

### 위기 대응 기록 양식

| 항목            | 내용                                       |
| :-------------- | :----------------------------------------- |
| **일시**        | YYYY-MM-DD HH:MM                           |
| **사용자 ID**   | (마음온 내부 ID)                           |
| **감지 등급**   | Level 1 / 2 / 3                            |
| **감지 키워드** | (AI가 감지한 키워드)                       |
| **담당자**      | (대응 담당자 성함)                         |
| **대응 조치**   | (전화 / 문자 / 방문 / 전문 연계)           |
| **대응 결과**   | (정상 확인 / 전문 연계 / 긴급 출동 / 오탐) |
| **메모**        | (특이사항)                                 |

### 보고 주기

- **Level 3**: 대응 즉시 기관장에게 구두 보고 → 24시간 내 서면 기록
- **Level 2**: 주간 회의 시 보고
- **Level 1**: 월간 통계에 포함

## 10. 면책 고지

> **마음온의 AI 위기 감지 기능은 보조적 수단이며, 전문적인 정신건강 평가를 대체하지 않습니다.**
> AI가 감지하지 못하는 위기 상황이 존재할 수 있으며, 최종 판단과 대응 책임은 현장 담당자와 전문 의료진에게 있습니다.
> 긴급한 상황에서는 즉시 119 또는 1393에 직접 연락해 주세요.
