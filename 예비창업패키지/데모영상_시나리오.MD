## A-4. 질문 카드 작성 + 음성 입력 (0:45 - 1:10)

**촬영 순서:**

1. (0:45) 첫 번째 질문 카드: **"잠은 잘 주무셨나요?"**
   - 텍스트 필드 탭 → "어젯밤 새벽 3시까지 잠이 안 왔어요" 입력
2. (0:50) 두 번째 질문 카드: **"오늘 무슨 일이 있었나요?"**
   - 🎤 **마이크 버튼 탭** → 음성 녹음 시작 (빨간 점 애니메이션)
   - 화면에 실시간 음성→텍스트 변환 표시
   - 말하기: "팀 회의에서 2주간 준비한 기획안을 발표했는데 팀장님이 방향이 다르다고 했어요"
   - 마이크 버튼 다시 탭 → 녹음 중지, 텍스트 확정
3. (1:00) 세 번째 질문 카드: **"어떤 감정이 들었나요?"**
   - 미리 복사해둔 텍스트 붙여넣기: "허탈하고 속상했어요. 나만 혼자 열심히 한 것 같아서요."
4. (1:05) 나머지 카드(선택)는 빠르게 스크롤하여 넘기기

**자막:**

- (0:45) "4단계 질문 카드로 감정을 구조화하여 기록합니다"
- (0:50) "🎤 음성 입력 지원 — 말하면 실시간으로 텍스트 변환됩니다"
- (1:00) "감정을 글로 표현하기 어려우면, 말로 편하게 기록할 수 있습니다"

> **촬영 팁:** 음성 입력 장면이 핵심! 마이크 버튼 탭 → 빨간 점 → 말하기 → 텍스트 자동 변환되는 흐름을 선명하게 촬영
