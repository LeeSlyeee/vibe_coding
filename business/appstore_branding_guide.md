## 3. 앱 설명문 (첫 3줄이 핵심)

### 변경 전

```
마음온은 AI 기반 정신건강 관리 앱입니다.
우울증, 불안장애 등 정신건강 문제를 조기에 감지하고...
```

### 변경 후

```
오늘 당신의 감정 온도는 몇 도인가요? 🌡️

매일 1분, 한마디로 기록하는 나만의 감정 일기.
AI가 당신의 이야기를 조용히 듣고, 감정의 흐름을 시각화해 드립니다.

✨ 주요 기능
• 한마디 감정 기록 — AI와 2~3문장 대화로 오늘의 마음 기록
• 마음 온도 트래커 — 감정 변화를 온도 그래프로 한눈에
• 주간/월간 리포트 — AI가 분석한 나의 감정 패턴
• 보호자 연결 — 소중한 사람과 마음 온도 공유

🔒 프라이버시
• 감정 대화는 폰 안에서만 처리 (On-Device AI)
• 서버에 대화 내용이 전송되지 않음

📞 위급 시 도움
• 자살예방상담전화 1393 (24시간)
• 정신건강위기상담 1577-0199
• 가까운 정신건강복지센터 검색 기능 내장
```

## 4. 스크린샷 전략

| 순서 | 내용                | 톤                    |
| :--- | :------------------ | :-------------------- |
| 1    | 마음 온도 메인 화면 | 따뜻한 일상           |
| 2    | 한마디 대화 화면    | "오늘 하루 어땠어요?" |
| 3    | 감정 캘린더         | 색깔로 보는 한 달     |
| 4    | 주간 리포트         | 감정 패턴 시각화      |
| 5    | 보호자 연결         | 소중한 사람과 공유    |

> ⚠️ 스크린샷에 "위기 감지", "자살 예방", "정신건강" 단어가 직접 노출되지 않도록 주의
> → 이 기능들은 앱 설명 하단 기능 목록에만 간략 표기

## 5. 키워드 전략

### 사용할 키워드 (일상 프레이밍)

- 감정 일기, 마음 일기, 기분 기록, 감정 온도, 마음 온도
- 셀프 케어, 하루 기록, 일기 앱, 감정 트래커
- AI 일기, 대화형 일기

### 사용하지 않을 키워드 (직접적 의료 용어)

- ~~정신건강~~, ~~우울증~~, ~~불안장애~~, ~~심리 상담~~
- ~~자살 예방~~, ~~위기 감지~~

> 이 키워드들은 ASO 관점에서 검색 유입에 불리할 수 있으나,
> **낙인 효과 최소화 → 다운로드 전환율 향상**이라는 트레이드오프를 선택
