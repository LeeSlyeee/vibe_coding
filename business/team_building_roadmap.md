# 마음온(Maum-On) 팀 빌딩 로드맵 (P3-수정11)

> **작성일**: 2026-03-02
> **근거**: 투자자·정치인·장관 3개 이해관계자가 공통 지적한 "1인 팀 리스크"
> **목표**: 예비창업패키지 수료 → 초기창업패키지 진입까지의 팀 확장 계획

---

## 1. 현재 팀 구성

| 역할             | 인원 | 비고                         |
| :--------------- | :--: | :--------------------------- |
| 창업자 (CEO/CTO) | 1명  | 풀스택 개발 + AI + 사업 총괄 |

**리스크**: Bus Factor = 1. 창업자 부재 시 서비스 전면 중단 가능

---

## 2. Phase 1: 예비창업패키지 기간 (2026 Q1~Q2)

### 목표: 핵심 인력 1~2명 확보

| 우선순위  | 포지션         | 역할                                  | 채용 방식                   | 예상 시기 |
| :-------: | :------------- | :------------------------------------ | :-------------------------- | :-------- |
| **1순위** | RA/임상 전문가 | IRB 승인, 파일럿 설계, PHQ-9 상관분석 | 파트타임 자문 → 정규직 전환 | 2026 Q2   |
| **2순위** | BD/사업개발    | B2G 영업, 보건소·복지센터 관계 구축   | 공동창업자 또는 정규직      | 2026 Q2   |

### 채용 채널

- 🏫 **대학 연구실**: 정신건강 분야 석·박사 과정생 (RA 역할)
- 🏥 **정신건강복지센터 OB**: 현장 경험 + 네트워크 보유자
- 💼 **예비창업패키지 네트워킹**: 동기 창업팀 간 co-founding
- 🌐 **링크드인/원티드**: BD 포지션 공개 채용

### 비용 계획 (예비창업패키지 예산 내)

| 항목                 | 월 비용 | 기간  | 소계          |
| :------------------- | :------ | :---- | :------------ |
| RA 자문료 (파트타임) | 100만원 | 6개월 | 600만원       |
| BD 인건비 (정규직)   | 250만원 | 4개월 | 1,000만원     |
| **소계**             |         |       | **1,600만원** |

---

## 3. Phase 2: 파일럿 완료 후 (2026 Q3~Q4)

### 목표: 핵심 팀 3~4명 + 자문단

| 포지션                | 역할                                      | 조건              |
| :-------------------- | :---------------------------------------- | :---------------- |
| **프론트엔드 개발자** | iOS/Android 네이티브 앱 유지보수          | 정규직            |
| **디자인 리서처**     | UX 리서치, 사용자 인터뷰, UI 개선         | 파트타임 → 정규직 |
| **임상 자문단**       | 정신건강의학과 전문의 1인, 임상심리사 1인 | 월 1회 자문 계약  |

### 자문단 구성안

| 분야                  | 역할                                    | 빈도     |
| :-------------------- | :-------------------------------------- | :------- |
| 정신건강의학과 전문의 | 위기 감지 알고리즘 검증, 임상 근거 자문 | 월 1회   |
| 임상심리사            | AI 대화 시나리오 감수, PHQ-9 해석       | 월 2회   |
| 법률 자문             | 개인정보보호, 의료기기법 사전 검토      | 분기 1회 |

---

## 4. Phase 3: 초기창업패키지 진입 시 (2027 Q1~)

### 목표: 팀 5~7명 규모

```
CEO/CTO ─── 기술 총괄
    ├── 프론트엔드 개발자
    ├── AI/ML 엔지니어 (On-Device AI 최적화)
    ├── BD/사업개발
    ├── RA/임상 전문가
    └── 디자인 리서처

[자문단]
    ├── 정신건강의학과 전문의
    ├── 임상심리사
    └── 법률 자문
```
