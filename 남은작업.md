# 마음 브릿지 (Mind Bridge) 남은 작업

> 작성일: 2026-03-06  
> 코드 작업은 모두 완료. 아래는 **수동 작업**만 남음.

---

## 1. App Store Connect — 구독 제품 등록

- [ ] App Store Connect → 내 앱 → 구독 메뉴 진입
- [ ] 구독 그룹 생성: `Mind Bridge`
- [ ] 제품 등록:
  - `mindbridge_monthly` — 월간 구독 (₩4,900 또는 원하는 가격)
  - `mindbridge_yearly` — 연간 구독 (₩39,000 또는 원하는 가격)
- [ ] 각 제품에 표시 이름, 설명 입력
- [ ] **참고 파일**: `ios_app/MindBridgeSubscription.storekit` (로컬 테스트용 StoreKit 설정)
- [ ] `project.yml`의 StoreKit Configuration 경로가 올바른지 확인

## 2. Google Play Console — 구독 제품 등록

- [ ] Google Play Console → 수익 창출 → 구독 메뉴 진입
- [ ] 구독 제품 생성:
  - `mindbridge_monthly` — 월간 구독
  - `mindbridge_yearly` — 연간 구독
- [ ] 기본 요금제(Base Plan) 설정
- [ ] **참고 파일**: `android_app/gradle/libs.versions.toml` (Billing 라이브러리 버전)
- [ ] `android_app/app/src/main/java/com/maumon/app/data/billing/SubscriptionManager.kt` 에서 제품 ID 일치 확인

## 3. 실기기 E2E 테스트
