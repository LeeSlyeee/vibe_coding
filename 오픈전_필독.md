## ⚠️ 1. 구독 우회 (Beta Free Access) 원복

### 현재 상태

베타 테스트 기간이므로 **마음 브릿지 구독 없이도 모든 기능이 무료로 사용 가능**하도록 수정됨.  
`SubscriptionManager`가 항상 `isSubscribed = true`를 반환하며, 실제 StoreKit/BillingClient의 구독 확인을 건너뜀.

### 원복해야 할 파일 (4곳)

#### 📱 iOS: `ios_app/SubscriptionManager.swift`

| #   | 위치                              | 현재 (베타)                                | 원복 (정식)                                 |
| --- | --------------------------------- | ------------------------------------------ | ------------------------------------------- |
| 1   | `isSubscribed` 초기값             | `@Published var isSubscribed: Bool = true` | `@Published var isSubscribed: Bool = false` |
| 2   | `updateSubscriptionStatus()` 함수 | 베타 코드 활성 + 원본 주석 처리            | 베타 코드 삭제 + `/* */` 주석 해제          |

- 파일 내에서 `[BETA]` 키워드로 검색하면 수정 위치를 바로 찾을 수 있음
- `updateSubscriptionStatus()` 안에 `/* ── 정식 출시 시 아래 코드로 교체 ──`로 시작하는 주석 블록이 원본 코드

#### 🤖 Android: `android_app/app/src/main/java/com/maumon/app/data/billing/SubscriptionManager.kt`

| #   | 위치                              | 현재 (베타)                     | 원복 (정식)                        |
| --- | --------------------------------- | ------------------------------- | ---------------------------------- |
| 1   | `_staticIsSubscribed` 초기값      | `MutableStateFlow(true)`        | `MutableStateFlow(false)`          |
| 2   | `_isSubscribed` 초기값            | `MutableStateFlow(true)`        | `MutableStateFlow(false)`          |
| 3   | `updateSubscriptionStatus()` 함수 | 베타 코드 활성 + 원본 주석 처리 | 베타 코드 삭제 + `/* */` 주석 해제 |

- 마찬가지로 `[BETA]` 키워드로 검색하면 3곳 모두 바로 찾을 수 있음

### 영향 범위

구독 우회로 인해 아래 화면/기능이 **무료로 열려 있음**:

| 화면             | iOS 파일                   | Android 파일              | 영향                          |
| ---------------- | -------------------------- | ------------------------- | ----------------------------- |
| 감정 리포트 공유 | `AppShareView.swift`       | `ShareScreen.kt`          | Paywall 없이 바로 Export 진입 |
| 캘린더 넛지 배너 | `MoodCalendarView.swift`   | `CalendarScreen.kt`       | 구독 유도 배너 표시 안 됨     |
| 설정 - 멤버십    | `AppSettingsView.swift`    | `SettingsScreen.kt`       | "구독 중" 상태로 표시됨       |
| 마음 브릿지 메인 | `MindBridgeMainView.swift` | `MindBridgeMainScreen.kt` | 구독 없이 바로 진입 가능      |
